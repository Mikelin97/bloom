# Bloom Social Reading Club - Progress Log

## Codebase Patterns

### Existing Patterns (discovered from codebase)
- Firebase Auth via `src/contexts/AuthContext.tsx` - use `useAuth()` hook
- Reader context at `src/context/ReaderContext.tsx` - scroll position, book state
- Interaction context at `src/context/InteractionContext.tsx` - personas, messages
- API base from `import.meta.env.VITE_API_BASE`
- Glassmorphism: `bg-white/10 backdrop-blur-xl border border-white/20`
- Server runs on port 8787, proxied via Vite

### New Patterns (add as discovered)
- Room/session shared contract in `src/types/readingSession.ts`
- Server room store in `server/rooms.js` (in-memory, max participants = 6)
- Book metadata + paragraph parsing in `server/books.js`
- RAG utilities in `server/moderator.js` with embedding cache fallback when no API key
- Socket event contract uses: `room:join`, `room:state`, `chat:send`, `chat:message`, `chat:typing`, `presence:update`, `reader:position`
- Room APIs: `/api/books`, `/api/rooms`, `/api/rooms/:roomId`, moderator APIs under `/api/moderator/*`
- New social routes in `src/App.tsx`: `/rooms`, `/room/:roomId`
- UI theme toggle pattern via `src/hooks/useUiTheme.ts` with `localStorage` key `bloom_ui_theme`
- Custom test localStorage shim centralized in `src/test/setup.ts`
